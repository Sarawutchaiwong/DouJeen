# Project Instructions: Chinese Discovery Game

This project follows the **Impeccable** design-first methodology. All development must align with the vision in `PRODUCT.md` and the visual system in `DESIGN.md`.

## Core Directives
1. **Design First:** Do not write UI code without first checking `DESIGN.md`.
2. **Cute & Clean:** Maintain the "Kawaii Alchemist" aesthetic. Avoid cluttered layouts or corporate styles.
3. **Discovery Focus:** Every feature should contribute to the "Learn through discovery" goal.

## Impeccable Vocabulary
Use these keywords to guide the AI's refinement of the UI:

- **`polish`**: Review the current UI and apply subtle improvements to spacing, shadows, and animations to match the "Bubbly Exploration" style.
- **`audit`**: Check the current implementation against `PRODUCT.md` and `DESIGN.md` and list any discrepancies.
- **`bolder`**: Increase visual weight, contrast, or vibrancy of key elements.
- **`quieter`**: Reduce visual noise, lighten colors, or increase whitespace to focus on core content.

## Technical Constraints
- **Framework:** Next.js 16.2.9 (Custom version, see `AGENTS.md`).
- **Styling:** Vanilla CSS (as per Gemini CLI defaults for rich aesthetics) or Tailwind if already configured (currently Tailwind 4 is in `package.json`).
- **Icons:** Prefer stylized SVG shapes over standard icon libraries to maintain the "Cute" style.
